# HR 경영진 Agent (v1 · MVP)

경영진 회의 중 **권한에 맞는 인사 데이터를 정확히 즉시** 제공하는 대시보드형 HR AI 비서.
좌측 **대시보드** + 우측 **챗봇** 복합 화면, 본부장별 **Row-level Security** 권한 분리, 업로드 데이터에 **그라운딩된 답변**(환각 차단)이 핵심입니다.

## 빠른 시작 (로컬)

```bash
npm install
npm run dev          # http://localhost:3000
npm run build && npm start   # 프로덕션 모드
```

## 데모 계정 (로그인 화면에서 선택)

| 계정 | 역할 | 열람 범위 |
| --- | --- | --- |
| 김영업 | 영업본부장 | 영업본부(SALES) 데이터만 |
| 박지원 | 경영지원본부장 | 경영지원본부(MGMT) 데이터만 |
| 최연구 | 연구개발본부장 | 연구개발본부(RND) 데이터만 |
| 이관리 | HR 시스템 관리자 | 업로드/감사로그 전용 · 개별 본부 데이터 조회 불가 |

본부장 A로 로그인하면 본부 B의 데이터는 대시보드·챗봇 어디에서도 노출되지 않습니다(핵심 성공기준).

## 주요 기능 (Phase 1)

- 역할 기반 로그인 + 부서코드 기반 행 단위 접근통제(`lib/access.ts`)
- 대시보드: 퇴사위험·지시사항·인건비·채용·초과근무·승진·PIP·다면평가·연령구성비
- 챗봇: 대표 질문 10선 룰 기반 RAG 답변, 임원 보고체, 출처(사내/외부) 명시
- 관리자: 엑셀/CSV 수동 업로드(데모), 적재 현황, 감사 로그
- 데이터에 없는 질문 → "사내 데이터에 없음 + 일반 트렌드 참고" 형식으로 분리 안내

## 데이터

- 데모 더미 데이터: `lib/hr_data.json` (실제 인사정보 아님)
- HR 담당자용 표준 업로드 템플릿: `templates/HR데이터_업로드템플릿_v1.xlsx` (샘플 채움) / `_빈양식.xlsx`

## 기술 스택

Next.js 14 (App Router) · React 18 · TypeScript · Recharts. 외부 백엔드/DB 없이 클라이언트에서 동작하는 MVP로, Vercel에 그대로 배포 가능합니다.
